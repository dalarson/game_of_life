# game_of_life

An interactive implementation of Conway's game of life. A front end built with TypeScript and Angular to facilitate user interaction. A backend built in rust to control game logic. An API built with WebSockets for real time updates to the game.

# To build server

`cargo b`
`cargo r`
A random board will be generated to start. Press enter in the terminal to proceed to the next generation.
